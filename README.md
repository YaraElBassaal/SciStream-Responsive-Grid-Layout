# 🔬 SciStream – Responsive Grid Layout


---

**SciStream** is a responsive science news layout built using **HTML5 and CSS Grid**.  
The project focuses on creating a modern, flexible UI that adapts seamlessly across different screen sizes using advanced layout techniques.

## 🎯 Design Requirements 👍
The SciStream project aims to build a clean and responsive news layout using **CSS Grid**.  
The design requirements include:

- Structuring the entire page using CSS Grid  
- Creating a responsive multi-column layout  
- Using **grid-template-areas** for better layout control  
- Adapting layout across different screen sizes (mobile → tablet → desktop)  
- Displaying categorized science articles in a clear visual hierarchy  

---

## 🛠️ Technologies Used 👌
The project was built using:

- HTML5  
- CSS3  
- **CSS Grid Layout** (main focus 🔥)  
- Google Fonts (Manrope & Source Code Pro)

---

## ✨ Features 😎
SciStream includes the following features:

- Fully responsive layout using CSS Grid  
- Dynamic grid structure with **grid-template-areas**  
- Clean and modern UI design  
- Article cards with images and categories  
- Hover and focus states for better UX  
- Mobile-first design approach  
- Adaptive layout for:
  - Small screens (single column)  
  - Medium screens (two columns)  
  - Large screens (complex grid layout)
